## 📈 Excel用数式サンプル

### Excelに貼り付け後、以下の数式を設定

#### 進捗率の自動計算
```
=IF(L列>0, (L列-N列)/L列*100, 0)
```

#### ステータスの色分け条件付き書式
- **完了**: 緑色
- **進行中**: 黄色
- **未着手**: グレー
- **保留**: 赤色

#### 残時間の自動計算
```
=L列-M列
```

#### 週次合計工数
```
=SUMIF(ステータス列, "完了", 実績列)
```

## 💡 使い方のコツ

### 1. WBS番号の振り方
- 1, 2, 3... (大分類)
- 1.1, 1.2... (中分類)
- 1.1.1, 1.1.2... (小分類)

### 2. 依存関係の記載
- 前のタスクのWBS番号を記載
- 例: 1.1.3 → 「1.1.1」(Git installが完了しないと始められない)

### 3. 見積もりの精度向上
- 初回は実績を記録
- 2回目以降は実績を参考に見積もり
- バッファ20-30%を含める

### 4. 定期更新のタイミング
- **毎朝**: 今日のタスクの確認
- **15:00**: 進捗率の更新
- **退勤前**: 実績時間の記録
- **週末**: 週次サマリーの作成

## 📊 サマリービュー作成例

### 週次サマリー用数式

**総見積時間**:
```
=SUM(L:L)
```

**総実績時間**:
```
=SUM(M:M)
```

**完了率**:
```
=COUNTIF(I:I,"✅完了")/COUNTA(I:I)*100
```

**稼働率**:
```
=総実績時間/総見積時間*100
```
